# novels-kingdom

منصة عربية ثابتة لعرض الروايات، تسجيل القراء والكتّاب، التعليقات، ولوحة إدارة مرتبطة بـ Supabase عند النشر.

## إعداد Supabase

1. افتح مشروع Supabase المرتبط بالموقع.
2. من **SQL Editor** نفّذ الملف `supabase/schema.sql` كاملًا لتجهيز الجداول، السياسات، والدوال.
3. أنشئ حساب المدير من واجهة الموقع أو من Supabase Auth.
4. من جدول `profiles` غيّر قيمة `role` لحساب المدير إلى `admin`.
5. تأكد أن `assets/supabase-config.js` يحتوي على رابط المشروع ومفتاح publishable فقط.

> لا تضع مفاتيح `service_role` أو كلمات مرور قاعدة البيانات داخل ملفات الواجهة.

## الملفات المهمة

- `assets/app.js`: طبقة البيانات، المصادقة، التخزين المحلي الاحتياطي، والمزامنة مع Supabase.
- `assets/supabase-config.js`: إعدادات Supabase العامة للمتصفح.
- `supabase/schema.sql`: مخطط قاعدة البيانات وسياسات RLS.
- `SECURITY.md`: ملاحظات أمنية قبل النشر العام.
